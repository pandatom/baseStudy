

# [MyBatis 处理sql中的 大于，小于，大于等于，小于等于](https://www.cnblogs.com/liangliping/p/10184919.html)

Mybatis中的sql语句中的 “<” 和  “>” 号要用转义字符  “`<`”  和 ”`>`“ ，否则会报错！

如查找年龄大于等于指定年龄的用户信息：

```sql
SELECT * FROM T_USER WHERE AGE &gt;=#{age}
```

进行转义：

 

| 原符号   | <    | <=    | >    | >=    | &     | '      | "      |
| -------- | ---- | ----- | ---- | ----- | ----- | ------ | ------ |
| 替换符号 | &lt; | &lt;= | &gt; | &gt;= | &amp; | &apos; | &quot; |
